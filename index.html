netDx

netDx is a patient classifier algorithm that can integrate several types of patient data into a single model. It does this by converting each type of data into a view of patient similarity; i.e. by converting the data into a graph in which more similar patients are tightly linked, while less similar patients are not so tightly linked. netDx is now in BioConductor! Go here to see vignettes (worked examples) and install software. Reference: Methods paper:Pai S, Hui S, Isserlin R, Shah MA, Kaka H, GD Bader (2019) netDx: interpretable patient classification using integrated patient similarity networks.
